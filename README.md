# College Chatbot

This is a simple, command-line chatbot for a fictional college. It can answer questions about admissions, academic programs, contact information, tuition, financial aid, housing, clubs, and sports.

## How to Run

1.  **Navigate to the project directory:**
    ```bash
    cd college_chatbot
    ```

2.  **Run the chatbot:**
    ```bash
    python3 main.py
    ```
    Alternatively, you can use the provided `run.sh` script from the root directory:
    ```bash
    ./run.sh
    ```

3.  **Interact with the chatbot:**
    - Type your questions at the prompt (`>`).
    - Type `exit` to quit the application.

## How to Run Tests

To run the automated tests, navigate to the `college_chatbot` directory and run the following command:

```bash
python3 test_chatbot.py
```
